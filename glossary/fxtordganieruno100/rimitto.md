# 指値注文（リミット注文）

{% hint style="info" %}
エントリーと利確時の決済に使用。

決済時に含み損の価格でも現在価格の条件次第で使用可。
{% endhint %}

あらかじめ注文を入れる価格を指定し予約する注文方法のこと。

「買い」の場合には現在より安い価格で、「売り」の場合は現在より高い価格を指定し注文を入れることができる。

決済で使う場合は、「買い」「売り」に関わらず、利益確定を行う価格をあらかじめ指定しておく時に使います。
