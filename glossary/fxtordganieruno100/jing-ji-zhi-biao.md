# 経済指標

{% hint style="info" %}
代表的な発表内容：雇用数、失業率、GDP、インフレ率、小売売上高、日銀短観、貿易・サービス収支など
{% endhint %}

政府や世界各国の中央銀行の発表する経済の成長や安定性などの指標となる経済データのこと。注目度の高い発表ほど為替の価格変動に大きな影響を与える。

また、各国の中央銀行の総裁や財務大臣などが公の場で金融政策について発言することを「要人発言」と呼び、経済指標と同様に為替の価格変動に大きな影響を与えることがある。
