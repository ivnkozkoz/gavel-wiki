# 逆指値注文（ストップ注文）

{% hint style="info" %}
エントリーと損切り時の決済に使用。

決済時に含み益の価格でも現在価格の条件次第で使用可。
{% endhint %}

注文方法の概要は指値注文と同じだが、指定できる価格が指値注文とは反対で

「買い」の場合には現在より高い価格で、「売り」の場合は現在より安い価格を指定し注文を入れることができる。

決済で使う場合は、「買い」「売り」に関わらず、損切りを行う価格をあらかじめ指定しておく時に使います。
