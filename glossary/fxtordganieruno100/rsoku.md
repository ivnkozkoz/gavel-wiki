# ローソク足

{% hint style="info" %}
その他では、「バーチャート」や「ラインチャート」がある。
{% endhint %}

明治30年頃に日本で開発された日本で最も人気なチャートタイプ。他にも「バーチャート」や「ラインチャート」があるが、日本人のほとんどはローソク足を使っている。特別な理由がなければローソク足を使うのがよい。

また、ローソク足の四角で囲まれている部分を「実体」と言い、線になっている部分を「ヒゲ」と呼ぶ。
