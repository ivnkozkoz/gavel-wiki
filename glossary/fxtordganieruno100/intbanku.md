# インターバンク市場（銀行間市場）

{% hint style="info" %}
主な参加者：中央銀行、為替ブローカー、証券会社、短資会社、電子ブローキング
{% endhint %}

金融機関などの限られた参加者同士が電話やコンピュータなどの通信インフラを用いて通貨の取引を行う市場のこと。\
最低でも、100万ドル単位で取引が行われる。場合によっては、数億ドルの取引が行われる。
