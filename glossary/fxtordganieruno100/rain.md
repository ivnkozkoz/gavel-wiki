# 水平線（水平ライン）

{% hint style="info" %}
多くのレジスタンスラインはサポートラインに、サポートラインはレジスタンスラインに役割が転換する。
{% endhint %}

トレーダーが強く意識し抵抗がみられる価格帯に引かれる水平なライン。\
上方向の値動きに対する抵抗線をレジスタンスラインと呼び、\
下方向の値動きに対する抵抗線をサポートラインと呼ぶ。
